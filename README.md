# Customer Churn Prediction API 🚀

A Machine Learning based Customer Churn Prediction system that predicts whether a customer is likely to leave a service. The project uses a trained ML model, FastAPI for API deployment, Swagger UI for testing, and MLflow for experiment tracking.

---

## 📌 Project Overview

Customer churn prediction helps businesses identify customers who are likely to stop using their services. 

This project builds an end-to-end Machine Learning pipeline:

- Data preprocessing
- Feature engineering
- Model training
- Model evaluation
- MLflow experiment tracking
- FastAPI deployment
- Swagger API documentation

---

## 🚀 Features

✅ Machine Learning based churn prediction  
✅ FastAPI REST API  
✅ Interactive Swagger UI documentation  
✅ MLflow model tracking  
✅ Model saved using Joblib  
✅ Real-time customer prediction  
✅ Production-ready API structure
